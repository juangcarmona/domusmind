## 現実

ユキは晴人のサッカー練習が何時に終わるか知っている。
木曜日の家庭訪問を覚えている。
牛乳を買わないといけないことを把握している。
モモの獣医の予約が来週あることも知っている。

ユキが家のことを考えるのをやめると、何かがうまくいかなくなる。

問題はユキが忙しいことではない。問題は、家がユキの記憶に依存していることだ。

## なぜツールが役に立たないか

タスクアプリはアイテムを記録する。カレンダーは時間を記録する。
どちらも責任を追跡しない。
誰が何を把握しているかを示さない。
家をシステムとしてモデル化しない。

家の状態は、いまも一人の頭の中にある。

## 変化

DomusMindは家族全員に家の状態を見えるようにする。

- 誰が家のどのエリアを担当しているか
- 今週何が起きるか
- どんなタスクがあり、誰のものか
- どんなルーティンが家を動かしているか

一人が全てを抱える必要がなくなる。

## 見えるもの

```
今日 — 佐藤家

晴人 → サッカー練習 17:30
ユキ → 家庭訪問 09:00

牛乳を買う
ランドセルを確認する
```

家族全員が同じ画面を見る。
予定、タスク、責任 — 一か所に。

## どう機能するか

1. 何かが追加される — イベント、タスク、ルーティン
2. 家族の記録の一部になる
3. 家族のタイムラインに表示される
4. 適切な人物と責任に紐づけられる

## 今あるもの

DomusMind V1には以下が含まれます：

- **家族** — 世帯のアイデンティティとメンバー
- **責任** — 誰がどのエリアを担当するか
- **イベント** — 家族カレンダーの予定と約束
- **タスク** — 何をすべきか、誰がするか
- **ルーティン** — 家を動かす定期的な作業
- **タイムライン** — 起きていることすべての統合ビュー

## 誰のためか

DomusMindは、一人が全てを記憶することに依存するのをやめたい**家族**のために作られています。

**開発者**向け：DomusMindはオープンシステムです。APIはドキュメント化され、アーキテクチャは公開されており、貢献を歓迎します。
