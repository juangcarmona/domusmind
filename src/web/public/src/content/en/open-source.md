## Open platform

DomusMind is open household infrastructure, not a closed consumer app.

## How to contribute

Contribute through issues, discussions, pull requests, architecture feedback, and domain model improvements.

## Repository structure

The repository contains vision docs, architecture, domain definitions, implementation code, and tests.

## Roadmap direction

Roadmap evolves from foundational modeling to broader domains and intelligent assistance.
